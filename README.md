# Next JS

## About

This is my first mini-project in Next JS, also my first contanct with Next JS.

## procedures

### Install Next JS

<pre>
   npx create-next-app@latest
</pre>

or

<pre>
   yarn next-app
</pre>

or

<pre>
   pnpm next-app
</pre>

### After Install

<ul>
   <li>Run: 
      <pre>npm run dev</pre> or <pre>yarn dev</pre> or <pre>pnpm dev</pre> (start dev server)
   </li>
</ul>
